# HOW TU USE

# Compilation 
- Download make / sudo apt-get install make or https://sourceforge.net/projects/gnuwin32/
- Add to path 
- Open Powersheel / CMD
- cd directory
- make all 
- Run !!!

# How to run 
```
    ./read_rs232 <string> <parity> <stop>
```

|                |PARITY|STOP|
|----------------|-------------------------------|-----------------------------|
||`RS232_PARITY_WITHOUT`            |`RS232_STOP_BIT_ONE`
||`RS232_PARITY_PEER`            |`RS232_STOP_BITS_TWO`            
||`RS232_PARITY_ODD`|
